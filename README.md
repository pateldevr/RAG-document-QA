# RAG Document QA

A Streamlit web app that lets you upload a PDF and ask questions about it — answers are generated by an AI and grounded strictly in the document's content.

## What it does

1. **Upload** a PDF document
2. **Index** it — the app extracts text, splits it into chunks, and stores vector embeddings in a local database
3. **Chat** — ask any question about the document and get concise, context-grounded answers
4. **Reset** — clear the document and start over with a new one

Answers are generated only from the uploaded document. If the answer isn't in the document, the assistant will say so.

## How it works

```
PDF Upload
    │
    ▼
┌─────────────────┐
│  PDF Extraction │  indexing/pdf_extractor.py  — reads all pages via PyPDF
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│    Chunking     │  indexing/chunker.py        — splits text into 500-char chunks (100-char overlap)
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   Embedding     │  indexing/embedder.py       — encodes chunks with all-MiniLM-L6-v2
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   ChromaDB      │  vector_store/chroma_client.py — persists embeddings locally
└─────────────────┘

User Question
    │
    ▼
┌─────────────────┐
│ Query Embedding │  indexing/embedder.py       — same model encodes the question
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   Retrieval     │  query/retriever.py         — top-3 similar chunks fetched from ChromaDB
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  LLM Generation │  query/generator.py         — Google Gemini 2.5 Flash Lite answers from context
└────────┬────────┘
         │
         ▼
     Chat UI         app.py                     — Streamlit interface
```

## How to run it

### Prerequisites

- Python 3.10+
- A [Google Gemini API key](https://aistudio.google.com/app/apikey)

### Setup

```bash
# 1. Navigate to the project directory
cd "AI Learning/rag_document_qa"

# 2. Create and activate a virtual environment
python -m venv .venv
.venv\Scripts\activate        # Windows
# source .venv/bin/activate   # macOS / Linux

# 3. Install dependencies
pip install -r requirements.txt

# 4. Add your Gemini API key
#    Create a .env file in the project root:
echo GEMINI_API_KEY=your_key_here > .env
```

### Run

```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

### Usage

1. On first load, upload a PDF using the file picker and click **Index Document**
2. Wait for indexing to complete (a spinner will show progress)
3. Type questions in the chat input at the bottom
4. Click **Reset** (sidebar or top of page) to clear the document and chat history

## Project structure

```
rag_document_qa/
├── app.py                      # Streamlit UI and pipeline orchestration
├── requirements.txt            # Python dependencies
├── .env                        # API keys (not committed)
├── indexing/
│   ├── pdf_extractor.py        # Extract text from PDF pages
│   ├── chunker.py              # Split text into overlapping chunks
│   └── embedder.py             # Generate sentence embeddings (all-MiniLM-L6-v2)
├── vector_store/
│   └── chroma_client.py        # ChromaDB init, store, and retrieve operations
└── query/
    ├── retriever.py            # Embed query and fetch top-k relevant chunks
    └── generator.py            # Build RAG prompt and call Gemini API
```
